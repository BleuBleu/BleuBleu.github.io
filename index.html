FamiStudio

FamiStudio NES Music Editor

FamiStudio is very simple, DAW-style NES music editor. It is designed to be easier to use than FamiTracker, but its feature set is much more limited.

Download

FamiStudio 1.1.0 is available as Windows installer (.msi) and as a macOS application bundle.

Change Log

Version 1.1.0:

  • New export dialog
  • Added WAV export
  • Added FamiTracker Text export (see the wiki for limitations)
  • Added NSF export (see the wiki for limitations)
  • Bug fixes and code refactoring.
